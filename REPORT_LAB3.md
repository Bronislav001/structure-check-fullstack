# ЛР-3 — что реализовано

- Фильтрация, поиск, сортировка и пагинация истории проверок.
- CRUD для проверок: просмотр, редактирование и удаление.
- Загрузка файлов TXT / MD / PDF / DOCX до 5 MB.
- Извлечение текста из загруженного файла и автоматическая проверка структуры.
- Автоматическое прикрепление исходного файла к созданной записи.
- Скачивание по подписанной ссылке и удаление вложений.
- Ограничение доступа по ролям: пользователь работает со своими данными, manager / admin — по своим правам.
